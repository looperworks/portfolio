# LooperWorks Portfolio

Source code for [looperworks.com](https://looperworks.com) — the portfolio site for LooperWorks, a design and research practice operating at the intersection of architecture, urbanism, and education.

## About

LooperWorks engages built work, speculative projects, and pedagogical programming — developing spatial propositions grounded in material discipline and critical inquiry. Founded in 2019, the practice collaborates with students, educators, and design organizations to advance rigorous work across scales and contexts.

## Deployment

This site is deployed via [GitHub Pages](https://pages.github.com/) with a custom domain (`looperworks.com`).
